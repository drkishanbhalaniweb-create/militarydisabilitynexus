## Project Soul

**What is this project?**
Military Disability Nexus: A premium web platform helping veterans secure VA disability benefits. It includes specialized services like Nexus Letters, DBQs, and C&P Exam Coaching. The stack is Next.js with Tailwind CSS, utilizing Supabase for backend functions (Auth, DB, Edge Functions) and Stripe + Cal.com for payments and scheduling.

**Key Principles:**
- **Premium Aesthetics ("vfresh")**: Interfaces must be modern, use generous whitespace, feature curved edges (`rounded-[2.5rem]`/`rounded-3xl`), and utilize subtle gradients or shadows.
- **HIPAA/Security First**: All medical-related integrations and data handling must fall under secure processes.
- **Standalone Conversion Flows**: Campaign landing pages must be self-contained (e.g., Lead Capture -> Payment -> Scheduling).
- **Code Modularity**: Rely heavily on Shadcn UI and reusable lib functions (e.g., `payment.js`).

## Key Decisions

- **C&P Coaching Payment Flow**: Switched from traditional consulting booking to a "payment-first" direct Stripe integration with a dedicated Cal.com success page.
- **Edge Function Modifications**: Updated the generic `create-checkout-session` Supabase Edge Function to directly accept a `priceId` to handle hardcoded standalone products like the $150 C&P Coaching service.
